# Hangman Game

A fun and colorful Hangman game built with React — my very first React project, made as a school assignment!

## About the Project

This is a classic Hangman game where you try to guess a hidden word one letter at a time before the figure is fully drawn. The project was a great introduction to React fundamentals like components, state management, and event handling.

## Features

- Random word selection on each new game
- Visual hangman figure that builds up with each wrong guess
- Wrong letter tracking
- Win/lose popup with a replay option
- Notification for already-guessed letters
- Pink pastel design

## Built With

- [React](https://react.dev/)
- CSS3

## Getting Started

```bash
# Clone the repo
git clone https://github.com/tsuikaaa/hangman-react.git

# Install dependencies
npm install

# Run the app
npm start
```
